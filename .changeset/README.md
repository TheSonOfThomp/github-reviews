# Changesets

This directory contains changeset files generated by `@changesets/cli`.

## Adding a changeset

When making a code change in a PR, run:

```
pnpm changeset
```

Follow the prompts to select a bump type (major/minor/patch) and describe the change.

## Release process

1. PRs with changesets are merged into `main`
2. The GitHub Actions workflow automatically opens a "Release" PR that bumps versions
3. When the Release PR is merged, the extension is built and published to the Chrome Web Store
